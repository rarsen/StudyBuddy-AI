‚ùì TROUBLESHOOTING:

   Issue: "OpenAI API error"
   ‚Üí Check API key in .env file
   ‚Üí Verify OpenAI account has credits
   ‚Üí Restart backend: docker-compose restart backend

   Issue: "Database connection failed"
   ‚Üí Check database is running: docker-compose ps db
   ‚Üí Wait longer (database takes ~10 seconds to start)
   ‚Üí Check DATABASE_URL in .env

   Issue: "Port already in use"
   ‚Üí Kill process: lsof -ti:8000 | xargs kill -9
   ‚Üí Or: lsof -ti:5173 | xargs kill -9

   Issue: "Frontend won't load"
   ‚Üí Check frontend logs: docker-compose logs frontend
   ‚Üí Verify VITE_API_URL points to http://localhost:8000
